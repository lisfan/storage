Storage

new Storage()

离线存储控制器

  1. storage只会管理存储通过其 API 创建的离线数据，会加入命名空间前缀，不管理其他自定义的存储数据
  2. storage实例会单独建立一个映射表来管理数据单元与数据的映射关系
