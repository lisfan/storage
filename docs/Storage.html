Constructor

new Storage(options)

Source:
  • storage.js, line 36
  • storage只会管理存储通过其 API 创建的离线数据，不管理其他自定义的存储数据（数据项会加上命名空间前缀）
  • storage实例会单独建立一个映射表来管理数据单元与数据的映射关系
